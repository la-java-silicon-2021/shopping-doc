# G101：商品一覧画面

## 画面レイアウト

![](image/G101_商品一覧画面.svg)

## 画面項目定義

|  No.  |  項目  |  I/O  |  備考  |
| :---: | --- |:---:| --- |
| 1 | 全商品リンク | - |  |
| 2 | カテゴリリンク | O | 存在するカテゴリをすべて出力 |
| 3 | 検索入力ボックス | I |  |
| 4 | 検索ボタン | - |  |
| 5 | 商品画像 | O |  |
| 6 | 商品名 | O |  |
| 7 | 価格 | O |  |
| 8 | カート追加ボタン | - |  |
| 9 | 戻るボタン | - |  |
| 10 | 確認ボタン | - |  |

## イベント

- 全商品リンク押下
    - すべての商品情報を本画面に出力する

- カテゴリリンク
    - 選択したカテゴリの一覧情報を本画面に出力する

- 検索ボタン押下
    - 検索入力

- 確認ボタン押下
    - 会員登録確認画面に遷移する
    - 入力内容にエラーがあった場合はエラーメッセージを表示する
