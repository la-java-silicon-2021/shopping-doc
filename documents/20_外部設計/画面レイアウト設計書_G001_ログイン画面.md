# G001：ログイン画面

## 画面レイアウト

![](image/G001_ログイン画面.svg)

## 項目定義

|  No.  |  項目  |  I/O  |  備考  |
| :---: | --- |:---:| --- |
| 1 | エラーメッセージ | O | ログイン処理にてエラーが発生した場合出力 |
| 2 | メールアドレス | I |  |
| 3 | パスワード | I |  |
| 4 | ログインボタン | - |  |
| 5 | 新規会員登録リンク | - |  |

## イベント

- ログインボタン押下
    - 入力されたメールアドレスとパスワードでログインできた場合は商品情報一覧画面に遷移する
    - ログインできなかった場合はエラーメッセージを①のメッセージ項目に出力する

- 新規会員登録リンク押下
    - 新規会員登録画面に遷移する
